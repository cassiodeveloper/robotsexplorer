Robots Explorer

Ferramenta para checagem de Directory Listing

Download as .zip Download as .tar.gz View on GitHub

Bem vindo

Esta página foi criada para apresentar melhor a ferramenta Robots Explorer.

Features

  • Request para um domínio para get do robots.txt;
  • Leitura e interpretação (parse) do arquivo;
  • Apresentação amigável do parse;
  • Listagem dos diretórios marcados como 'disallow';
  • Opção de exploitar cada diretório marcado como 'disallow';
  • Exibição dos resultados de exploit para cada diretório;
